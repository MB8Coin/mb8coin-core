This is the working repository for  MB8Coin

For more information please visit the website:

https://mb8coin.io

## Coin Specifications

| Specification | Value |
|:-----------|:-----------|
| Block Spacing | `2 minutes` |
| Stake Minimum Age | `4 hours` |
| Stake Reward | `5% per annum (10% for the first 6 months)` |
| Port | `38380` |
| RPC Port | `38381` |
